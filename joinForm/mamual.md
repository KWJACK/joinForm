# mysql+nodejs로 만든 게시판
- 이미지업로드가 가능한 게시판
- jquery-ui 적용
- 게시물 하나당 이미지 업로드는 1개뿐(nosql로 가야할듯)
- 페이지 화면 구성이 로그인에 따라 변함
- 세션을 통한 암호화. passport, bkfd2Password 모듈 적용
